### Java-calculator

## Chapter1. 
  - 기본적인 사칙연산 가능.
  - 단순 양의 정수와 연산자를 입력 받아 사칙연산을 하고 결과를 프린트 하는 작업을 exit를 입력 할때 까지 반복합니다
  - 예외처리는 하지않아 -1 같은 음수 혹은 문자열 입력 시 프로그램이 종료 됩니다

## Chapter2.
  - calculator class 를 생성하여 객체를 활용해 사칙연산을 수행 하는 프로그램
  - 기본적인 동작은 Chapter1과 동일
  - 추가 기능
    - 지금 까지 계산 결과를 저장하는 기능 추가
    - 지금 까지 계산 결과 중 가장 첫번째로 저장된 기록을 제거하는 기능 추가( 예외처리 x)

## Challenge.
  - 연산자 (+,-,*,/) 를 enum 타입으로 받을 수 있도록 하여 타입 안정화
  - Calculator 클래스에 제네릭을 적용하여 int 뿐만 아니라 double 타입으로 선언하여 객체를 활용 할 수 있도록 변경
  - 지금 까지 저장된 데이터 중 새로이 입력받는 수 comparisonNumber 를 입력받아 해당 수 보다 큰 수를 stream 을 이용하여 구현
  - 사용자가 메뉴를 통해서 원하는 기능을 선택하여 수행 할 수 있도록 변경
